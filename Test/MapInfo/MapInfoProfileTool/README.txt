Overview
The profile tool allow you to take elevation profiles from grids of type: asc, txt, dem, img, grd, flt and dt*. 
These can be loaded into MapInfo through the File > Open menu where types of file is set to "Grid"

Requirements
The tool does NOT require Vertical mapper or Discover and is designed for those without these extentions.

Requires MapInfo 11.5+ and .NET Framework 4

Source
The code is open souce and is all hosted at GitHub
